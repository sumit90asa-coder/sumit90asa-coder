## 🤖 How I Use AI to Build Faster

I don't just use AI as a chatbot. I use it as a development accelerator:

- **Architecture review** — describe the system, ask it to find weaknesses before writing code
- **Prompt-driven debugging** — structured root cause analysis instead of random Googling
- **Rapid prototyping** — generate boilerplate with full context, review, understand, adapt
- **Documentation** — auto-generate docstrings and README sections after building features

This is how I shipped a full-stack platform in 48 hours at a hackathon.
